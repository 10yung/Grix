Grix

Eazy, Lazy, Crazy lightway Flex-box Grid System

Grix is a 12 columns (in each row) Grix system, you can use 1 ~ 12 columns as you like. Each column has 10px left and right padding.

You can use outer div to control columns inside each row, so you don't have to type one by one when you create inner div.

If the class has gx-* its for parent <div> , and without that prefix it's for child <div>

                        
<-- This means two columns in one row, each 50% width -->
<div class="gx-row gx-cols-2 gx-colml-2">
    <div>
        <h5>6</h5>
    </div>
    <div>
        <h5>6</h5>
    </div>
</div>

Grix have some class like gx-fh gx-center fill fill-space sf-cen to let you control layout

1. You can use gx-fh to create full screen height.

2. gx-center to center child <div>

3. fill to fill space when you don't have enought child <div>

4. fill-space will fill with equally space wrapper (if you dont have enought child <div>)

5. sf-cen to make child <div> self center, always used in full page center.

This is full screen height with self-center child <div> 

                    
<div class="gx-vert gx-center gx-fh">
    <div class="sf-cen">
        <h3>Full page centered !!! </h3>
    </div>
</div>

Full page centered !!!

Grix also support vertical arragement. If you use gx-vert you can make child <div> stack from top to down. And you can use gx-fh to make it full screen height


Grix cut screen height into 9 sections.
And you can use vh-* to set child <div> height. If you use vh-1 means the height of that child <div> is 1/9 screen height.

Once you use vh-* to arrange vertical object, Grix also has vertical offset v-ofset-* to let you easier to control your layout.
